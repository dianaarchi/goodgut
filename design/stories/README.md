# design/stories

JSX components for Instagram Stories (9:16 format).

4 story types × 3 layout variants = 12 components:
- Hook   — grabs attention, teases the carousel
- Fact   — single punchy gut health stat or claim
- CTA    — drives follow/save/share action
- Engage — poll, question sticker, interactive prompt

Rendered to PNG via Satori + sharp during the pipeline run.
