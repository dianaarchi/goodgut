# design/carousel

JSX components for Instagram carousel posts (1:1 format, 8 slides).

Brutalist acid minimal aesthetic — Anton + Space Mono fonts, black/off-white base,
rotating 5-accent palette (acid green, electric cyan, Klein blue, violet, fuchsia).

These components are the **source of truth** for the visual system.
Rendered to PNG via Satori + sharp during the pipeline run.
